#Jarvis
An Virtual AI Assitant for your daily work

## 🚀 Features
*   **Voice Control**
*   **Weather Updates**
*   **Music Library**: "Add your fav music in music library.songs dictionary with links"
*   **App Launcher**:It can launch apps on your voice command add you app name and path in launcher.launchapps dictionary
*   **LLM**:Install ollama and llama3.2:1b model locally

🛠️ Installation & Setup

Follow these steps to get a local copy up and running:

### 1. Clone the Repository
```bash
git clone https://github.com
cd JARVIS
```

### 2. Set Up a Virtual Environment
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Configure Environment Variables
Create a `.env` file in the root directory by copying the example template:
```bash
cp .env.example .env
